# react-native-beacon-mgr

a beacon manager for react native

## Installation

```sh
npm install react-native-beacon-mgr
```

## Usage

```js
import BeaconMgr from "react-native-beacon-mgr";

// ...

const result = await BeaconMgr.multiply(3, 7);
```
